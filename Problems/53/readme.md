# Maximum Subarray

| Solution Idx | Time Complexity | Space Complexity | Comment |
| ------------ | --------------- | ---------------- | ------- |
| 1            | O(n)            | O(n)             | DP      |
| 2            | O(n)            | O(1)             | DP      |

## DP

- Prefix Sum
- Stack to find previous min element ( pop non-viable solution )
- End at index i subarray to consider all solution
