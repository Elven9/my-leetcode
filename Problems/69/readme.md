# Sqrt(x)

| Solution Idx | Time Complexity | Space Complexity | Comment                                            |
| ------------ | --------------- | ---------------- | -------------------------------------------------- |
| 1            | O(n)            | O(1)             | Naive solution, search all solutions from 0 to n/2 |
| 2            | O(logn)         | O(1)             | Binary search on solution space, start from n/4    |
